# Flashcard_quiz_App

A command-line flashcard quiz application written in Python.

- **Quiz Mode** — Get tested on your cards with randomized questions
- **Add Cards** — Create your own question/answer flashcards
- **View Cards** — See all cards along with your accuracy stats
- **Delete Cards** — Remove cards you no longer need
- **Track Progress** — Each card tracks how many times you've answered correctly
- **Reset Stats** — Start fresh whenever you want

Run- python flashcard.py

1. Run the program
2. Choose **Start Quiz** to test yourself
3. Choose **Add Flashcard** to create new cards
4. Your accuracy stats are saved automatically after every quiz

- Cards are stored in a plain JSON file so no database setup is required
- Answers are checked case-insensitively for fairness
- Stats persist between sessions so you can track long-term progress
- Sample cards are included so you can try a quiz immediately
